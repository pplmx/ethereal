# ✨ Ethereal 开发路线图

> 以太之灵 - 智能桌面精灵助手

## 🎯 项目概述

- **技术栈**: Tauri v2 + Rust + React 19 + TypeScript + Ollama
- **核心特性**: 透明桌宠 + 硬件监控 + AI 交互
- **当前版本**: 0.1.0
